## sgutil get serial

sgutil get serial [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA/ACAP serial numbers.


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA/ACAP Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get serial -d 1**
