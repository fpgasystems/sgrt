## sgutil build mpi

sgutil build mpi [--help]

  &nbsp; &nbsp; Generates MPI binaries for your projects.



### Flags
-p, --project

  &nbsp; &nbsp; Specifies your MPI project name.


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil build mpi**
