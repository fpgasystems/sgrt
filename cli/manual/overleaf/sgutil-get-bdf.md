## sgutil get bdf

sgutil get bdf [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA/ACAP Bus Device Function.


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA/ACAP Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get bdf -d 1**
