## sgutil get platform

sgutil get platform [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA/ACAP platform names.


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA/ACAP Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get platform -d 1**
