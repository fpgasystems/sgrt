## sgutil get platform

sgutil get platform [flags] [--help]

  &nbsp; &nbsp; Platform names.


### Flags
-d, --device 

  &nbsp; &nbsp; Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get platform -d 1**
