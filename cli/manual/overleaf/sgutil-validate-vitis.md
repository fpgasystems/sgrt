## sgutil validate vitis

sgutil validate vitis [flags] [--help]

  &nbsp; &nbsp; Validates Vitis workflow on the selected FPGA.


### Flags
-d, --device 

  &nbsp; &nbsp; Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use Vitis validation.


### Examples
* **$ sgutil validate vitis**
