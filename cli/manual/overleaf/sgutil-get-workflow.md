## sgutil get workflow

sgutil get workflow [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA current workflow.


### Flags
-d, --device 

  &nbsp; &nbsp; Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get workflow -d 1**
