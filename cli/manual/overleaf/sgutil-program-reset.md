## sgutil program reset

sgutil program reset [flags] [--help]

  &nbsp; &nbsp; Resets a given FPGA/ACAP.


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA Device Index (see sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil program reset -d 1**
