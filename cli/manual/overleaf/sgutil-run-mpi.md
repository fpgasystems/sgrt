## sgutil run mpi

sgutil run mpi [--help]

  &nbsp; &nbsp; Runs your MPI application according to your setup.


### Flags
-p, --project

  &nbsp; &nbsp; Specifies your MPI project name.


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil run mpi**
