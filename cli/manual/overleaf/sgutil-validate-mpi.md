## sgutil validate mpi

sgutil validate mpi [flags] [--help]

  &nbsp; &nbsp; Validates MPI.


### Flags
-p, --processes 

  &nbsp; &nbsp; Specify the number of processes to use.


-h, --help 

  &nbsp; &nbsp; Help to use MPI validation.


### Examples
* **$ sgutil validate mpi -h**
* **$ sgutil validate mpi**
* **$ sgutil validate mpi -p 5**
