## sgutil build hip

sgutil build hip [flags] [--help]

  &nbsp; &nbsp; Generates HIP binaries for your projects.


### Flags
-p, --project 

  &nbsp; &nbsp; Specifies your HIP project name.


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil build hip -p hello_world**
