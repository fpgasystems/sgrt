## sgutil get name

sgutil get name [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA/ACAP device names.


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA/ACAP Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get name -d 1**
