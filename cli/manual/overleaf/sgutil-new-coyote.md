## sgutil new coyote

sgutil get coyote [--help]

  &nbsp; &nbsp; Creates a new project using Coyote Hello, world! template.



### Flags

  &nbsp; &nbsp; This command has no flags.


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil new coyote**
