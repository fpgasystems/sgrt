## sgutil validate hip

sgutil validate hip [flags] [--help]

  &nbsp; &nbsp; Validates HIP on the selected GPU.


### Flags
-d, --device 

  &nbsp; &nbsp; GPU Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use HIP validation.


### Examples
* **$ sgutil validate hip**
