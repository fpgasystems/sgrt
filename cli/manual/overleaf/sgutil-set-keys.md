## sgutil set keys

sgutil set keys [--help]

  &nbsp; &nbsp; Creates your RSA key pairs and adds to authorized_keys and known_hosts.


### Flags

  &nbsp; &nbsp; This command has no flags.


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil set keys -h**
* **$ sgutil set keys**
