## sgutil get network

sgutil get network [flags] [--help]

  &nbsp; &nbsp; Retreives FPGA/ACAP networking information. 


### Flags
-d, --device 

  &nbsp; &nbsp; FPGA/ACAP Device Index (according to sgutil examine).


-h, --help 

  &nbsp; &nbsp; Help to use this command.


### Examples
* **$ sgutil get network**
* **$ sgutil get network -d 0**
