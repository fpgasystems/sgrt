## sgutil update

<code>sgutil update [--help]</code>
<p>
  &nbsp; &nbsp; Updates sgutil to its latest version.
</p>

### Arguments
<p>
  &nbsp; &nbsp; This command has no arguments.
</p>

<code>-h, --help</code>
<p>
  &nbsp; &nbsp; Help to use this command.
</p>

### Examples

$ sgutil update
