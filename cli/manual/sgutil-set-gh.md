## sgutil set gh

<code>sgutil set gh [--help]</code>
<p>
  &nbsp; &nbsp; Enables GitHub CLI on your host.
</p>

### Flags
<p>
  &nbsp; &nbsp; This command has no flags.
</p>

<code>-h, --help <string></code>
<p>
  &nbsp; &nbsp; Help to use this command.
</p>

### Examples
```
$ sgutil set gh -h
$ sgutil set gh
```
