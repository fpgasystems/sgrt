## sgutil enable xrt

<code>sgutil enable xrt [flags] [--help]</code>
<p>
  &nbsp; &nbsp; Enables Xilinx Runtime (XRT).
</p>

### Flags
<code>-v, --version <string></code>
<p>
  &nbsp; &nbsp; XRT version.
</p>

### Examples
```
$ sgutil enable xrt
$ sgutil enable xrt --version 2022.1
```
